#Read before use
In this GitHub project you will find all the necessary things to run the tool.
This tool can be used to generate competitive teams of VGC, the current files allow to create teams for RegD.

To run the tool you will need to run it locally in your computer.
The creator used XAMPP so the rest of the ReadMe will be about how to use it with XAMPP.

To run this tool with XAMPP you will need to download XAMPP.
After doing so you will have a folder called XAMPP where you downloaded it. In it you will find another folder called htdocs move all the content of the project to this folder.
Then run the XAMPP application and click start on Apache.
Finally, just open your browser and search for http://localhost/index.php
